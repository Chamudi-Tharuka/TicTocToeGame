<h1 align="center"> Tic Toc Toe Game</h1>

<p align="center">
  
**A simple and interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. Users can choose to play as X or O, and play against a basic computer AI that makes random moves. <br>The game supports detecting wins, draws, and highlights winning combinations. It also includes a restart button to start a new game at any time. Designed with a modern, dark-themed UI and responsive grid layout.**

##  Features
**Player Selection** – Choose to play as X or O before starting the game.

**Interactive Game Board** – 3x3 grid with clickable cells to place X or O.

**Computer AI** – Plays against the user with random moves.

**Win Detection** – Automatically detects and displays the winner.

**Draw Detection** – Shows a draw message if all cells are filled without a winner.

**Restart Game** – Restart the game anytime using the restart button.

**Highlight Winning Cells** – The cells that form the winning combination are highlighted.

**Dynamic Player Turn** – Shows which player’s turn it is.

**Stylish UI** – Modern dark theme with hover effects, colorful X and O, and smooth transitions.

**Responsive Design** – Works well on different screen sizes.
